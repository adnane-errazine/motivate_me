## Quick Start

This project uses **`uv`**.

1.  **Install `uv`**:
    * `curl -LsSf https://astral.sh/uv/install.sh | sh` (macOS/Linux)
    * `irm https://astral.sh/uv/install.ps1 | iex` (Windows PowerShell)
    * pip install uv

2.  **Clone & Run**:
    ```bash
    git clone https://github.com/adnane-errazine/motivate_me
    cd motivate_me
    uv sync
    uv run app.py
    ```
